## CROSS-REFERENCE: I4 (2.0L) vs V6 (2.7L) Connector Code Mapping

| Function | 2.0L Code | 2.7L Code | Difference |
|---|---|---|---|
| ECM | C33 (single) | C133-1 through C133-5 | V6 has 5 ECM connectors vs 1 |
| TCM | C36-1/2/3 | C136-1/2/3 | Same structure, different prefix |
| Crankshaft position sensor | C13 | C113 | +100 offset |
| Camshaft position sensor | C14 | C114 | +100 offset |
| Throttle position sensor | C12 | C112 | +100 offset |
| Coolant temp sensor | C11 | C111 | V6 adds "& sender" |
| Oil pressure switch | C15 | C115 | +100 offset |
| Knock sensor | C23 (single) | C123-1, C123-2 | V6 has 2 knock sensors |
| Oxygen sensor (Up) | C16 | C116 (front) + C145/C146 (B1/B2 S1) | V6 has bank-specific codes |
