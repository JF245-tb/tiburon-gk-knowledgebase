## Key Component Locations (Engine Harness — CL-10+)

| Component | Location | Notes |
|-----------|----------|-------|
| ECM (Siemens SIMK43) | Behind passenger kick panel | C133-1 and C133-2 connectors |
| CKP sensor | Lower engine block, near crank pulley | C109 area |
| CMP sensor | Front of engine, cam journal area | 3-pin connector |
| MAP sensor | Intake manifold | 3-pin connector |
| TPS | Throttle body | 3-pin connector |
| ECT sensor | Coolant passage on engine | 1 or 2-pin connector |
| IAT sensor | Intake air duct | 2-pin connector |
