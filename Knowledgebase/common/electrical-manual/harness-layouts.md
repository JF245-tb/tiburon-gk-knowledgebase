## How to Use

Each page shows a harness as a line diagram with connector codes (IS/xxx, Mxx, Gxx) placed at their physical positions along the harness. Use these codes to cross-reference:

- **CC section** (`connector-configurations.md`) — get the pin assignments for that connector
- **CL section** (`component-locations.md`) — get the physical location on the car
- **SD section** (`schematic-diagrams.md`) — find the connector in a circuit schematic

---

## Connector Code Types

| Prefix | Type |
|--------|------|
| `IS/` + number | Inspection / splice connector |
| `M` + number | Junction / multiplex connector |
| `G` + number | Ground point |
| `C` + number | Named component connector (ECM, BCM, sensor) |
