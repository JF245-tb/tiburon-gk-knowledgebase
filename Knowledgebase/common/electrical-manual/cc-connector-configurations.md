# CC Section -- Connector Configurations: Complete Extraction

> **Source:** Electrical Troubleshooting Manual, CC section (CC.pdf, 24 pages)
> **Extracted:** 2026-03-12 via visual OCR from 300 DPI page images
> **Purpose:** Pin count, part number, and face drawing reference for every connector in the ETM CC section.
> **Part number format:** `CR##X###` where `##` = housing cavity count, `X` = type (F=female, M=male, B=board/module, T=terminal), `###` = design ID.
> **Caveat:** The housing cavity count in the part number does not always match populated pin count. X-marks in face drawings indicate unpopulated cavities.

## Part Number Format Reference

Hyundai ETM connector part numbers follow the pattern: **`CR##X###`**

| Field | Meaning |
|---|---|
| `CR` | Connector Reference prefix |
| `##` | Housing cavity count (not always = populated pins) |
| `X` | Type: **F** = female, **M** = male, **B** = board/module, **T** = terminal |
| `###` | Unique design/variant number |

**Important:** The cavity count in the part number represents the total housing size, not necessarily the number of populated pins. Connectors with X-marks on the face drawing have unpopulated cavities. Always cross-reference the face drawing for actual pin count.
