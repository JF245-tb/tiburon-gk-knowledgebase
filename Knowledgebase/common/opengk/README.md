# OpenGK Wiki — Local Reference
**Full wiki:** https://opengk.org/index.php?title=Main_Page
**Discord:** Available via OpenGK.org

OpenGK's goal is to open source the Hyundai/Kia Siemens ECMs to provide tuning options for DIY enthusiasts.
