Remove the multifunction switch and disconnect the wire connectors. Check the switch for continuity between the terminals. If continuity is not as specified, replace the wiper and washer switch.

## Front Wiper Motor

### Removal

1. Remove the wiper arm.

> **NOTE**
> Care must be taken not to scratch the engine hood.

<!-- Figure: Wiper arm removal from windshield, source: BE.pdf page 84 -->

**Tightening torque:** 7-11 Nm (70-110 kg-cm, 5.1-8.0 lb-ft)

2. Remove the weatherstrip and cowl top cover after removing the fasteners.

<!-- Figure: Weatherstrip and cowl top cover removal, source: BE.pdf page 84 -->

3. Disconnect the windshield wiper motor connector and remove the 3 bolts holding the windshield wiper motor and linkage assembly.

<!-- Figure: Wiper motor connector disconnection and 3-bolt removal, source: BE.pdf page 84 -->

**Tightening torque:** 22-29 Nm (230-290 kg-cm, 16.9-20.6 lb-ft)

4. Installation is the reverse of removal.

### Installation

1. Install the wiper arm to the specified position.

| Specified position | A     | B     |
|--------------------|-------|-------|
| Distance (mm)      | 30-40 | 30-40 |

<!-- Figure: Wiper arm installation position showing measurement distances A and B from windshield edge to blade tip, source: BE.pdf page 84 -->

2. Set the washer nozzle on the specified spray position.

<!-- Figure: Washer nozzle spray position adjustment with nozzle detail, source: BE.pdf page 84 -->

### Inspection

#### Speed Operation Check

1. Remove the connector from the wiper motor.

2. Attach the positive (+) lead from the battery to terminal 3 and the negative (-) lead to terminal 4.

3. Check that the motor operates at low speed.

4. Connect the positive (+) lead from the battery to terminal 3 and the negative (-) lead to terminal 5.

5. Check that the motor operates at high speed.

**Connector [D07]:**

```
  [D07]
  +--+--+--+--+--+
  | 1| 2| 3| 4| 5|
  +--+--+--+--+--+
```

| Pin | Function |
|-----|----------|
| 1   | Ground   |
| 2   | Parking  |
| 3   | B(+)     |
| 4   | Low      |
| 5   | High     |

<!-- Figure: Wiper motor connector D07 pinout for speed operation check, source: BE.pdf page 85 -->

#### Automatic Stop Operation Check

1. Operate the motor at low speed using the stalk connector.

2. Stop the motor operation anywhere except at the off position by disconnecting terminal 4.

3. Connect terminals 3 and 4.

4. Connect the positive (+) lead from the battery to terminal 3 and the negative (-) lead to terminal 1.

5. Check that the motor stops running at the off position.

**Connector [D07]:**

```
  [D07]
  +--+--+--+--+--+
  | 1| 2| 3| 4| 5|
  +--+--+--+--+--+
```

<!-- Figure: Wiper motor connector D07 for automatic stop operation check, source: BE.pdf page 85 -->

```
  +--+
```

# Sun Roof

## Sun Roof Switch

### Inspection

1. Disconnect the negative (-) battery terminal.

2. Detach the lamp lens then remove the mounting screws (2EA) and open the overhead console case cover then remove the mounting screws (2EA).

#### Connector [MM6] Continuity Table

| Position             | 1 | 2 | 3 | 7 | 8 |
|----------------------|---|---|---|---|---|
| Slide switch - Open  | O |   | O |   |   |
| Slide switch - Close |   | O | O |   |   |
| Tilt switch - Up     |   |   |   | O | O |
| Tilt switch - Down   |   |   |   | O | O |

O in the same row indicates continuity between those terminals. Polarity reverses between Up and Down positions for the tilt switch.

<!-- Figure: Sun roof switch connector MM6 continuity table, source: BE.pdf page 93 -->

3. After disconnecting the roof wiring and sun roof wiring connector then remove the overhead console lamp assembly from the headliner.

<!-- Figure: Overhead console removal showing roof wiring and sun roof wiring connector locations, source: BE.pdf page 93 -->

## Inspection of Components

### Head Lamp Relay

1. Remove the head lamp relay from the relay box in the engine compartment.

2. Check for continuity between terminals on the relay.

| Position          | 1 | 2 | 3 | 5 | A |
|-------------------|---|---|---|---|---|
| When de-energized | O |   |   | O |   |
| When energized    | O |   | O |   |   |

O in the same row indicates continuity between those terminals.

> **NOTE**
> 1. O---O: Indicates that there is continuity between the terminals.
> 2. Power symbol: Indicates that power is supplied.

<!-- Figure: Head lamp relay terminal continuity test, source: BE.pdf page 98 -->

### Hazard Switch

1. Disconnect the negative (-) battery terminal.

2. Remove the center facia panel with a flat-tip screwdriver.

<!-- Figure: Center facia panel removal for hazard switch access, source: BE.pdf page 98 -->

3. Detach the hazard lamp switch from the center facia panel.

#### Connector [M10] Pin Layout

```
  [M10]
  +--+--+--+--+--+--+
  | 1| 2| 3| 4| 5| 6|
  +--+--+--+--+--+--+
```

4. Operate the switch and check for continuity between terminals with an ohmmeter.

| Position | 1 | 2 | 3 | 4 | 5 | 6 |
|----------|---|---|---|---|---|---|
| OFF      |   |   |   |   |   |   |
| ON       | O |   | O |   | O | O |

O in the same row indicates continuity between those terminals.

<!-- Figure: Hazard switch connector M10 and continuity table, source: BE.pdf page 98 -->

### Rheostat

1. Disconnect the negative (-) battery terminal.

2. Disconnect the rheostat connector from the crash pad panel.

#### Connector [MN9] Pin Layout

```
  [MN9]
  +--+--+--+--+--+
  | 1| 2| 3| 4| 5|
  +--+--+--+--+--+
```

3. Check for intensity. If the light intensity of the lamps changes smoothly, without any flickering when the rheostat is turned, it can be assumed that the rheostat is normal.
