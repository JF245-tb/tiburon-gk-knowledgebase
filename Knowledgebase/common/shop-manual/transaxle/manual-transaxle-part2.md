## Clutch Housing

<!-- EMOB0560 -->

### Disassembly

<!-- EMOB0565 -->

1. Remove a shim of input rear and a oil channel.

2. Remove 2 bolts for the reverse bracket.

3. The bracket is raised up while having the reverse coupling by your hand and swing it.

4. You can remove those bracket easily with turning counter clockwise around no. idler shaft.

5. Then you can remove fork and bracket at same time.

6. Pay attention not to lose the cap.

<!-- Figure: Reverse bracket and shift fork removal, source: TR.pdf page 111 -->

7. Loosen the reverse check bolts and remove a spring, a rod, and a ball with a magnet tool.

8. The next, remove the reverse rod with other rods set in neutral position because of inter lock system (Refer to the "Manual transaxle shift control).

> **NOTE**
> It is easy to remove the reverse rod if you use a plastic hammer.

<!-- Figure: Reverse check bolt removal, source: TR.pdf page 111 -->

9. Loosen 5th & 6th shift check bolts and remove a spring, a rod, and a ball.

<!-- Figure: 5th & 6th shift check bolt removal, source: TR.pdf page 111 -->

10. Remove a pin for 1st & 2nd fork.

11. Hold the fork rod, pull out the fork while hitting by a plastic hammer, and remove a fork from the housing.

12. Pull out the bracket's pin.

13. Cover some clothes because it's easy to fly over the pin.

14. Remove C-ring which stops fork.

15. Remove a same C-ring in the lower side.

16. Pull out the fork rod.

17. If it is not easy to come off, pull out the interlock pin in the fork rod.

<!-- Figure: Fork rod and pin removal sequence, source: TR.pdf page 111 -->

18. Remove output shaft, input shaft, reverse idler gear and differential gear.

19. Pull out all check balls which remains in the housing.

<!-- Figure: Output shaft, input shaft, reverse idler gear removal, source: TR.pdf page 111 -->

20. Take off differential side bearing outer race and adjusting shim using special tool (09431-2C200).

<!-- Figure: Differential side bearing outer race removal with special tool 09431-2C200, source: TR.pdf page 112 -->

### Reassembly

<!-- EMOB0570 -->

1. Press on the differential oil seal using special tool (09431-2C000).

<!-- Figure: Differential oil seal installation with special tool 09431-2C000, source: TR.pdf page 112 -->

2. Press on the input oil seal using special tool (09431-2C100).

<!-- Figure: Input oil seal installation with special tool 09431-2C100, source: TR.pdf page 112 -->

3. Install the differential side bearing outer race using special tool (09431-0C300).

<!-- Figure: Differential side bearing outer race installation with 09431-0C300, source: TR.pdf page 112 -->

4. Install the oil channel.

<!-- Figure: Oil channel installation, source: TR.pdf page 112 -->

5. Install the output shaft front bearing and output shaft retainer.

6. Fix a bolt.

<!-- Figure: Retainer output shaft bearing installation, source: TR.pdf page 112 -->

7. Assemble the output shaft assembly, input shaft assembly, and reverse idler gear assembly to housing.

<!-- Figure: Shaft assemblies installation into housing, source: TR.pdf page 112 -->

8. Assemble the magnet to housing.

9. Assemble 1st-2nd bracket to a fork rod.

<!-- Figure: Fork rod assembly diagram showing 1st-2nd fork rod, 3rd-4th fork rod, 5th-6th fork rod, reverse fork rod, interlock sleeve, interlock ball, interlock pin, and check parts, source: TR.pdf page 113 -->

10. Insert 1st-2nd fork rod to a fork.

11. Assemble 1st-2nd fork&fork rod to 1st-2nd coupling and rod hole.

<!-- Figure: 1st-2nd fork rod insertion, source: TR.pdf page 113 -->

12. Set a hole position of 1st-2nd fork and 1st-2nd fork rod.

13. Assemble the pins.

<!-- Figure: Pin assembly for 1st-2nd fork, source: TR.pdf page 113 -->

14. Insert the inter lock pin to the fork rod.

15. Insert a 3rd-4th fork rod to a 3rd-4th fork and 3rd-4th bracket.

<!-- Figure: 3rd-4th fork rod insertion, source: TR.pdf page 113 -->

16. Assemble 3rd-4th fork&fork rod to 3rd-4th coupling and rod hole.

17. Fix 3rd-4th fork by C ring.

18. Set a hole position of 3rd-4th fork rod and 3rd-4th bracket.

<!-- Figure: 3rd-4th fork C-ring and positioning, source: TR.pdf page 113 -->

19. Pins are put in a hole on the same way to 1st-2nd fork rod.

<!-- Figure: Pin insertion for 3rd-4th fork, source: TR.pdf page 114 -->

20. Insert 2 interlock balls with the magnet tool.

<!-- Figure: Interlock ball insertion diagram showing 1st-2nd, 3rd-4th, 5th-6th, reverse fork rods, interlock sleeve, interlock ball, interlock pin, and check parts, source: TR.pdf page 114 -->

21. Install 5th-6th shift fork to 5th-6th coupling assembly.

<!-- Figure: 5th-6th shift fork installation, source: TR.pdf page 114 -->

22. Install the interlock pin to 5th-6th shift rod.

> **NOTE**
> Apply grease before putting inter lock pin. It's for falling off prevention.

23. Install 5th-6th fork rod to 5th-6th shift fork, and install 5th-6th fork rod to 3rd-4th shift fork hole. And the next, install 5th-6th bracket.

> **NOTE**
> Insert the fork rod which groove's direction keep (5th-6th check parts side).

<!-- Figure: 5th-6th fork rod installation, source: TR.pdf page 114 -->

24. Set the hole position of 5th-6th fork rod and 5th-6th shift fork.

25. Pin is put in hole on the same way to 1st-2nd fork rod.

<!-- Figure: 5th-6th fork rod pin positioning, source: TR.pdf page 114 -->

26. Fix 5th-6th bracket by 2 C-rings.

<!-- Figure: 5th-6th bracket C-ring installation, source: TR.pdf page 114 -->

27. Insert 2 inter lock balls with magnet tool.

28. Install the shift check ball, the shift check sleeve and the shift check spring, and then fix them by shift check plug. Refer to below instruction.

<!-- Figure: Shift check ball assembly, source: TR.pdf page 115 -->

29. Install the reverse bracket rod and the reverse bracket and put in order the reverse bracket direction.

30. Install the reverse shift fork to the reverse coupling assembly.

31. Install the shifter cap in the reverse shift fork and set them.

<!-- Figure: Reverse bracket and shift fork installation, source: TR.pdf page 115 -->

32. Install the reverse fork rod to reverse lever assembly.

33. Install the clutch housing while installing the reverse fork rod to the reverse shift fork.

34. Decide the knock pin's direction while pushing the reverse lever assembly, and put in it.

35. Fix 2 bolts.

<!-- Figure: Reverse fork rod and clutch housing installation, source: TR.pdf page 115 -->

36. Install the shift check ball, the reverse shift check sleeve and the reverse check spring, and then fix them by shift check plug. Refer to below instruction.

37. Confirm the action of interlock system.

<!-- Figure: Interlock system diagram showing 1st-2nd, 3rd-4th, 5th-6th, reverse fork rods, interlock sleeve, interlock ball, interlock pin, and check parts, source: TR.pdf page 115 -->

38. Install the input gear oil channel and the input gear bearing adjusting shim on input gear near bearing.

> **NOTE**
> It is easy to fall down the shim, so apply grease on the shim.

<!-- Figure: Input gear oil channel and adjusting shim installation, source: TR.pdf page 115 -->

39. Select the correct input gear adjusting shim of the input gear assembly to minimize the end play, and then put it.

End play (mm): 0 - 0.06

Shim thickness = L2 - L1 + end play

<!-- Figure: Input gear end play measurement, source: TR.pdf page 116 -->

## Reverse Idler Gear

<!-- EMOB0580 -->

### Components

<!-- EMOB0582 -->

<!-- Figure: Reverse idler gear exploded view showing: reverse idler adjusting shim, reverse idler & coupling sleeve hub set, reverse idler gear rear needle bearing, reverse rear thrust bearing, reverse insert spring, reverse synchronizer, reverse idler front gear, reverse idler gear front needle bearing, reverse front thrust bearing, reverse idler shaft, spring pin, source: TR.pdf page 118 -->

### Removal

<!-- EMOB0585 -->

1. Remove one shim.

<!-- Figure: Shim removal, source: TR.pdf page 119 -->

2. Remove the reverse idler rear gear, coupling, synchronizer ring and needle bearing.

<!-- Figure: Reverse idler rear gear disassembly, source: TR.pdf page 119 -->

3. Remove the reverse idler front gear, synchronizer ring and needle bearing.

<!-- Figure: Reverse idler front gear removal, source: TR.pdf page 119 -->

4. Pull out the spring pin.

<!-- Figure: Spring pin removal, source: TR.pdf page 119 -->

### Reassembly

<!-- EMOB0590 -->

1. Press on the spring pin in the reverse idler shaft.

<!-- Figure: Spring pin installation in reverse idler shaft, source: TR.pdf page 119 -->

2. The next, install 2 races and 1 thrust needle bearing.

3. Put on the dent of the reverse idler gear front thrust needle bearing.

4. Install the reverse idler shaft.

<!-- Figure: Thrust needle bearing and reverse idler shaft installation, source: TR.pdf page 119 -->

5. Install the reverse idler front needle bearing.

<!-- Figure: Front needle bearing installation, source: TR.pdf page 119 -->

6. Install the reverse idler front gear and the reverse synchronizer ring.

<!-- Figure: Reverse idler front gear and synchronizer ring showing reverse idler rear gear side and reverse idler front gear side, with reverse coupling sleeve, source: TR.pdf page 120 -->

7. Put on the top of the reverse coupling chamfer.

8. Keep on the top of the reverse insert springs notch, and put in wide area of a groove.

9. Install the other 2 reverse insert springs in the same way.

<!-- Figure: Reverse insert spring installation into coupling, source: TR.pdf page 120 -->

10. Install the reverse coupling sleeve as below figure.

<!-- Figure: Reverse coupling sleeve installation, source: TR.pdf page 120 -->

11. Confirm whether the chamfer and springs notch turns up, and then install a hub to them.

> **NOTE**
> Put 3 reverse insert springs in notch of hub.

<!-- Figure: Hub installation with springs, source: TR.pdf page 120 -->

12. Put in 2 races and 1 thrust needle bearing.

13. Put down a dent of the reverse idler gear rear thrust needle bearing.

<!-- Figure: Rear thrust needle bearing installation, source: TR.pdf page 120 -->

14. Install the reverse idler gear rear needle bearing in reverse idler shaft.

<!-- Figure: Rear needle bearing installation, source: TR.pdf page 120 -->

15. Set the hub notch and the reverse synchronizer ring notch, and install them.

> **NOTE**
> Pay attention not to fall the reverse insert spring.

<!-- Figure: Reverse idler rear gear and front gear assembly showing shim, source: TR.pdf page 121 -->

16. Put down the reverse idler coupling sleeve & hub set.

17. At last, install the reverse idler adjusting shim.

<!-- Figure: Reverse idler coupling sleeve and hub set installation, source: TR.pdf page 121 -->

18. Select the correct reverse idler adjusting shim to minimize clearance between the reverse idler gear and case, and then install it.

Allowance clearance (mm): 0.04 - 0.14

## Shift Lever (M/T)

<!-- EMOB0130 -->

### Removal

1. Remove the shift knob.

<!-- Figure: Shift knob removal, source: TR.pdf page 127 -->

2. Remove the spring and the cap (for 6-speed M/T).

<!-- Figure: Spring and cap removal (6-speed), source: TR.pdf page 127 -->

3. Remove the skirt (for 6-speed M/T).

<!-- Figure: Skirt removal, source: TR.pdf page 127 -->

4. Remove the console upper cover.

<!-- Figure: Console upper cover removal, source: TR.pdf page 127 -->

5. Remove the console assembly.

<!-- Figure: Console assembly removal, source: TR.pdf page 127 -->

6. Remove the shift cable clip.

<!-- Figure: Shift cable clip removal, source: TR.pdf page 127 -->

7. Remove the select cable clip.

<!-- Figure: Select cable clip removal, source: TR.pdf page 127 -->

8. Remove the snap pin of the select cable.

<!-- Figure: Select cable snap pin removal, source: TR.pdf page 127 -->

9. Remove the shift cable.

<!-- Figure: Shift cable removal, source: TR.pdf page 128 -->

10. Remove the shift lever assembly.

<!-- Figure: Shift lever assembly removal, source: TR.pdf page 128 -->

> **CAUTION**
> For 6-speed M/T, push the plate shaft outward with a driver or equivalent through the hole in the shift lever bracket to remove it.

<!-- Figure: 6-speed shift lever bracket removal procedure, source: TR.pdf page 128 -->

### Inspection

<!-- EMOB0140 -->

1. Check the select cable for proper operation and for damage.

2. Check the shift cable for proper operation and for damage.

3. Check the boot for damage.

4. Check each bushing for wear, abrasion, sticking, restricted movement or damage.

5. Check for a weak or damaged spring.

### Installation

<!-- EMOB0150 -->

1. Install the shift lever assembly.
